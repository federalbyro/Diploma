# NLLB Diplom Project
## v1.1

Небольшой проект для тестирования перевода описаний товаров через `facebook/nllb-200-3.3B`.

Сейчас основной сценарий такой:

- беру `.csv` с входными данными
- прогоняю через модель
- получаю перевод
- если есть `label_ru`, считаю метрики
- параллельно смотрю бенчмарки по скорости и нагрузке

Проект делается как каркас с заделом на будущее, чтобы потом можно было не переписывать всё с нуля, если я захочу:
- принимать файлы не только локально
- запускать это как сервис
- подбирать параметры модели автоматически
- гонять разные наборы данных и сравнивать результаты

---

## Что сейчас умеет проект

- загружать модель NLLB
- обрабатывать `.csv` файлы с текстами
- переводить в русский язык
- работать с test data
- считать метрики качества перевода
- считать простые бенчмарки по времени
- сохранять результат перевода
